# Codigo Raspberry





\# Sistema de Alarma con Raspberry Pi



\## Descripción del proyecto

Este proyecto consiste en el desarrollo de un sistema de alarma utilizando una Raspberry Pi y el lenguaje Python. 

El sistema detecta movimiento mediante un sensor PIR y activa diferentes avisos visuales y acústicos mediante LEDs y un zumbador.



\##  Funcionamiento

\- Sensor PIR para detección de movimiento.

\- LED amarillo como aviso inicial.

\- LED rojo como estado de alerta.

\- Zumbador que emite una señal acústica.

\- Introducción de contraseña para desactivar la alarma.



\##  Modificaciones realizadas

Durante el desarrollo se han realizado varias mejoras:

\- Mejora del patrón acústico del buzzer mediante pitidos cortos.

\- Implementación de bloqueo tras varios intentos fallidos.

\- Validación de contraseña para asegurar un formato correcto.



\##  Tecnologías utilizadas

\- Raspberry Pi

\- Python

\- Librería RPi.GPIO

\- GitHub para control de versiones
